# Cafe-de-Lune
Café de Lune es una cafetería artesanal que busca reflejar su esencia cálida, natural y acogedora mediante una página web moderna e intuitiva.   El sitio está desarrollado con **HTML**, **TailwindCSS** y **JavaScript**, y tiene como propósito transmitir la identidad del local y acercar su experiencia a nuevos clientes.
